Two Sum in Blind 75 (1)

Created2022-09-01|Updated2022-09-02|Leetcode(easy)
|Word count:788|Reading time:4min|Post View:

Two Sum

Description

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution , and you may not use the same element twice.

You can return the answer in any order.

Example 1

Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Example 2

Input: nums = [3,2,4], target = 6
Output: [1,2]

Example 3

Input: nums = [3,3], target = 6
Output: [0,1]

Constraints

  • 2 <= nums.length <= 10^4
  • -10^9 <= nums[i] <= 10^9
  • -10^9 <= target <= 10^9
  • Only one valid answer exists.

Thoughts

We can use a Hashmap to store the value and index of each element in the array. Then we can iterate through the array and check if the target minus the current element is in the Hashmap. If it is, we return the index of the current element and the index of the element in the hash table. If not, we add the current element and its index to the Hashmap.

Discussion

Q1. 為甚麼要把array的值放進map的key，而不是放進value呢？
A1. 因為map.find(a)可以在map中重複尋找為a的key，所以才把原陣列的值放進map的key，而不是放進value。

Q2. 為甚麼要用map呢？
A2. 因為map可以在O(1)的時間內找到key，所以才用map。

Tips & Concepts

  1. ans[0]為之前在map中找到的value，ans[1]為目前的i。(注意順序)
  2. for(const auto& s : range), 當你只想讀取range裡面的元素時，使用const auto& (range 可為array, vector, string, map, set等等)
  3. map的key為s.first, value為s.second

if in map,use s.first, s.second

if in vector,use s

  1. map的find()會回傳一個iterator，如果找不到，則會回傳map.end()。

  2. iterator的解構方式為(*it).first 或 it->first

  3. auto 關鍵字讓編譯器去推導該變數類型，所以不用寫出變數類型，但是要注意，auto 關鍵字只能用在變數宣告時，不能用在函式回傳值的宣告上。

  4. iterator(auto 等於 map<int,int>::iterator )向量 and map 可使用迭代器來呼叫其元素

  5. reverse iterator(auto 等於 map<int,int>::reverse_iterator )向量 and map 可使用反向迭代器來呼叫其元素
