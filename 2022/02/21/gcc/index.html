如何安裝好GCC?

Created2022-02-21|Updated2022-02-24|gcc
|Word count:418|Reading time:1min|Post View:

編譯器 (Compiler)

GNU Compiler Collection(GCC) Clang
支持編譯C、C++及GO等程式語言，目前最流行 的編譯器 Based on LLVM(Low Level Virtual Machine)，支援C、C++、Objective-C，more faster

GNU Compiler Collection(GCC)💥💥

1. 開放原始碼(open source)的跨平台compiler

2. Windows,Mac,Linux的安裝方法各異

Windows🪟 Mac Linux
可透過安裝Mingw(Minimalist GNU on Windows)安裝GCC 內建Clang，直接佔領gcc指令 使用指令安裝

Minimalist GNU on Windows - Mingw

  • 將GCC編譯器移植到Windows平台
  • 在官網下載


如何打開cmd🤔

在Windows打開cmd

在Mac打開terminal

在Mac上執行下列其中一項操作:

  • 按一下Dock中的 「啟動台」 圖像，在搜尋欄位中輸入 「終端機」 ，然後按一下終端機。🛎️
  • 在Finder中，打開 「應用程式/工具程式」 檔案夾，然後按兩下終端機。🛎️

如何測試GCC是否有裝好?

  1. 打開cmd or terminal
  2. 輸入 gcc -v
  3. 如果 安裝成功 ，可以看到安裝路徑和版本資訊

找不到GCC?

環境變數沒設定好 !!

  1. 點開環境變數，找到 變數path ，按編輯✅
  2. 在電腦中找到之前所下載的 mingw64/bin ，複製路徑，新增到環境變數中的path內✅

    cmd要記得重開 🔥

怎麼截圖?🤔

Shift+Windows+s

cmd+Shift+4

GCC指令-1

Why is -o in this code? 🙋

Answer:

-o < file > Write output to < file >

e.g. gcc helloworld.c -o helloworld.exe

GCC指令-2

Why is && in this code? 🙋

Answer:

This command will run the new program only if it compiles successfully.If there’s a problem with the compile,it will skip running the program and simply display the errors on the screen.

&& here means “and then if it’s successful,do this”
